Installation

  1. Install node.js
    (On Ubuntu I had to do: sudo apt-get install libssl-dev)
  2. Install Node Package Manager
  3. Install the necessary node modules
    npm install express stache now node-ffi
    Check that the NODE_PATH variable properly points to the directory where you installed the modules
  4. Checkout dashboardjs code
    git clone git://github.com/dmarcos/dashboardjs.git
  5. Run the node server
    cd dashboardjs
    sudo node server.js
  6. Test the server. Visit in your browser:
    http://localhost/dashboards/
