Installation

  1. Install the latest stable version of node.js
    (On Ubuntu I had to do: sudo apt-get install libssl-dev)
  2. Install Node Package Manager
  3. Install the necessary node modules
    npm install express@2.5.9 async underscore faye xml2js mysql@0.9.6 socket.io socket.io-client hat browserchannel coffee-script@1.2.0 connect@1.8.6
  4. Copy the modified stache module to the node/node_modules directory
  5. Checkout dashboardjs code in the node directory
    git clone git://github.com/dmarcos/dashboardjs.git
  6. Checkout ShareJS code in the node directory
    git clone git://github.com/dmarcos/ShareJS.git
  7. Run the node server
    cd dashboardjs
    sudo node server.js
  8. Test the server. Visit in your browser:
    http://localhost/dashboard/
