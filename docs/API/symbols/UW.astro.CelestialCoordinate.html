<static> {CelestialCoordinate} UW.astro.CelestialCoordinate.parseCoord(text)
Parses a string that represents a coordinate to return a CelestialCoordinate with the specified RA and DEC. The parser is supposed to be fairly tolerant, as it will be translating from a web text input. The input can be in the form of HMS or decimal degrees.
Parameters:
{String} text
Expects a coordinate repesented by RA and DEC in H M S or decimal degrees as
00 00 00.0 +00 00 00.0
or 00 00 00.0 -00 00 00.0
or 00:00:00.0 +00:00:00.0
or two decimal numbers
where 0 is any digit.
Decimals not required, but a + or - is required for the DEC.
Returns:
{CelestialCoordinate} A CelestialCoordinate if text string represents an RA and DEC otherwise returns null.

setColor(color)
Set the color of the point. This will also change the iconUrl to the specified color. The color is a string that is the color name. Supported colors are:
  • red
  • green
  • yellow
  • blue
All other colors will set the variable, but will set the iconUrl to the default icon.
