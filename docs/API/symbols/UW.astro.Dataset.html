Class Detail
UW.astro.Dataset(json)
Create a new Dataset.
Parameters:
{object[]} json
A JSON array from which the dataset is created. The JSON should be of the form of an array of objects where each property is a column name, and each value is the value of that row and column. First object in array is first row in dataset.
